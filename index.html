🤖 Como atualizar via Claude

Todas as carteiras em 3 passos
1
O arquivo JSON foi baixado
Ele contém todos os ativos de todas as suas carteiras (ações, FIIs, ETFs) agrupados por carteira, sem os preços preenchidos.
2
Abra o Claude e envie o arquivo
Vá em claude.ai, clique no clipe 📎, anexe o JSON e envie a mensagem abaixo:
Atualize as cotações dos ativos neste arquivo JSON com os preços atuais da B3. Preencha precoAtual e variacao para cada ativo em todas as carteiras e devolva o JSON completo atualizado. 📋 Copiar
3
Baixe o JSON e importe
O Claude devolve o JSON com todos os preços preenchidos. Salve o arquivo e clique em "📥 Importar do Claude" — todas as carteiras são atualizadas de uma vez.
💡 Cripto não entra no arquivo — continua sendo atualizada automaticamente via CoinGecko a cada 5 minutos, sem precisar fazer nada.
🔁 Frequência sugerida: Uma vez por dia, antes de abrir o dashboard. Leva menos de 1 minuto e atualiza tudo de uma vez.
